プリント内容と問題集語句を入れた暗記カード/PWAです。

## 使い方

- 最初は `4択ペア` で50問演習をやる。
- 間違えた語は `弱点` に入り、同じ回でもう一度出る。
- p.108-109対策は `語句入力`。
- 択一対策は `正誤仕分け`。ここは問題集レベルの誤文判定を多めにしています。

## 収録内容

- 問題集語句まで含めた用語カード196枚
- 環境条約
- 悪徳商法
- 景気名・戦後経済の補助語
- EU年表・地域統合
- 国際協力の略語と混同注意
- 最恵国待遇・内国民待遇・セーフガードなど問題集の識別語
- 減反・環境基本法・FIT/FIP・AIIB/一帯一路などの周辺語

## 正誤仕分けの強化

正誤仕分けは、カードの自動変換ではなく専用の正誤問題を出します。

- 役割を逆にした誤文
- 条約・制度の対象を入れ替えた誤文
- 年・順序・目標数を入れ替えた誤文
- 問題集で狙われやすい「もっともらしい誤り」
